# Vein Mine - Minecraft 模组

[English](readme.md) | 简体中文

## 简介

Vein Mine 是一个轻量、快速跟进新版本的 Minecraft 模组，可帮助玩家高效地采集相连的同类方块。

无论是挖矿、砍树、收获农作物，还是收集建筑材料，Vein Mine 都能让这一过程更加快速便捷，同时保持高度可配置，并对多人游戏环境友好。

与许多同类连锁采集模组不同，Vein Mine 提供了高级方块匹配选项，使玩家和服务器管理员能够更精确地控制哪些方块应被视为同一连锁目标。

## 功能

### 连锁采集

* 一次操作即可采集相连的同类方块。
* 支持原版方块和模组方块。
* 支持可配置的激活模式（切换模式 / 按住模式）。

### 高级方块匹配（1.4.1+）

* 区分成熟与未成熟作物。
* 区分不同颜色的染色方块。
* 区分深板岩矿石与普通矿石。
* 可选的跨单格间隙搜索。

### 工具保护

* 防止工具意外损坏。
* 可配置的耐久度保护阈值。
* 支持自定义受保护工具列表。

### 多人游戏友好

* 服务端统一管理配置。
* 客户端设置自动遵循服务器规则。
* 在多人游戏环境中保持一致的行为表现。

### 高亮预览

* 实时高亮显示目标方块。
* 在破坏前预览即将被连锁采集的方块。

## 配置

### 1.4.0+

使用默认快捷键打开配置界面：

`V + M`

### 旧版本

Fabric：

安装 **Mod Menu**，然后在 **Mods** 菜单中打开 Vein Mine。

NeoForge：

在 **Mods** 菜单中打开 Vein Mine。

### 手动配置

* 编辑：` .minecraft/config/veinmine.json `

* 对于专用服务端，配置文件位于：`<服务端目录>/config/veinmine.json`

* 使用 `/veinmine reload` 命令重新加载配置文件。

* 所有配置项的详细说明请参阅：

* [配置文档](https://github.com/diaoyugan/Veinmine/blob/master/config_docs/config_zh_cn.md)

## 控制

* `~`（默认）：切换 Vein Mine 开关
* `V + M`（默认）：打开配置界面（1.4.0+）

## 注意事项
